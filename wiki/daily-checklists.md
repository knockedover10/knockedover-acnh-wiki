## First-10-Days Checklist

A structured guide for the critical early window. Tick off as you go.

### Day 1
- [ ] Pick island (if not done) — river mouth placement matters for future layout
- [ ] Set up tent, talk to villagers
- [ ] Collect hardwood, softwood, stone, iron nuggets
- [ ] Catch fish and bugs to donate to Tom Nook
- [ ] Start paying off 2,000 NM tent fee

### Day 2
- [ ] Finish tent fee → upgrade to house
- [ ] Continue collecting and donating critters (need 5 total for Blathers)
- [ ] Begin gathering materials for upcoming builds

### Day 3
- [ ] Blathers' tent opens → donate fish, bugs, fossils
- [ ] Talk to Tom Nook about island development
- [ ] Obtain DIY recipes from villagers and balloon presents
- [ ] Dig fossils and donate all to Blathers

### Day 4
- [ ] Build the bridge (required for Nook's Cranny unlock)
- [ ] Set 3 villager plot locations — plan these carefully before placing!
- [ ] Collect more Nook Miles for NMT trips

### Day 5
- [ ] New villagers arrive → greet them
- [ ] Use NMT to hunt for wishlist villagers (Kabuki, Shino as priority)
- [ ] Museum should be building → check upgrade status

### Day 6
- [ ] Nook's Cranny should open → begin using shop daily
- [ ] Check Nook Shopping and Nook Stop for limited items
- [ ] Start placing outdoor furniture to build toward 3-star rating

### Day 7
- [ ] Mable should visit → buy from her stall
- [ ] Continue NMT island hopping if villager slots open
- [ ] Plant flower varieties — mix colours to start hybrid breeding

### Day 8
- [ ] Mable's second or third visit → Able Sisters unlock approaching
- [ ] Continue 3-star preparation: outdoor furniture, fencing, flowers, trees
- [ ] Check daily fossil spots (4 fossils per day)

### Day 9
- [ ] Able Sisters should open (if Mable visited enough)
- [ ] Continue pursuing wishlist villagers via NMT
- [ ] Plan out fencing zones — fencing contributes to island rating

### Day 10
- [ ] Review island rating with Isabelle (once she arrives) or self-assess
- [ ] Ensure 7+ villagers are on island or in process of moving in
- [ ] Begin rough layout planning for major districts (shrine, bamboo, onsen)
- [ ] See [Island Layout](island-layout.md) for district planning reference

## Reusable Daily Checklist

Run through this every day once the island is past early game.

**Morning**
- [ ] Check Nook Stop for daily Nook Miles bonus (500 NM)
- [ ] Talk to all villagers — gifts, new dialogue, friendship building
- [ ] Dig up 4 fossils → donate new ones, sell duplicates
- [ ] Check the beach for washed-up items and bottles (DIY recipes)
- [ ] Check balloon presents (listen for pop sound near cliffs/coast)

**Shops & Economy**
- [ ] Check Nook's Cranny for daily rotating items and turnip price
- [ ] Check Able Sisters for new clothing items
- [ ] Sell fish, bugs, and extra materials as needed

**Nature & Maintenance**
- [ ] Water flowers (or check if rain is forecast)
- [ ] Check for hybrid flower growth
- [ ] Clear any weeds
- [ ] Shake trees for furniture, wasps (net ready), and Bells

**Villager Interaction**
- [ ] Give a gift to 1–2 priority villagers (friendship farming)
- [ ] Complete any villager requests or errands

**Optional / As Needed**
- [ ] NMT island hopping for villager hunting
- [ ] Stalk Market — check AM and PM turnip prices
- [ ] Catch seasonal fish or bugs before they rotate out
