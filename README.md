> 2. **Keep this README's Contents list, Current Snapshot, and Recent Updates list current** — add an entry to Recent Updates for every meaningful change, and trim it to the last 5 entries.

> 5. **Append a short dated entry to [📋 Changelog](wiki/changelog.md)** summarising what changed.
